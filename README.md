# Overview
二开了该开源库作为毕设的一个后台，主要逻辑是根据ip定位地址，并尽可能多的收集客户端信息记录下来
架构为MVC,使用以下资源:
- Sqringboot
- mysql
- thymeleaf
- cesium
> 默认使用数据库sbap和client_info表，初始化会创建user和admin用户，密码也为admin
